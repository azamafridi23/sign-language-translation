# ASL (American Sign Language) Translation Web Application

## Overview

This is a web application that uses deep learning to translate American Sign Language (ASL) into english alphabets. The application is built using Flask, a Python web framework,OPENCV, a Python framework for computer vision related tasks and leverages deep learning models to interpret and generate ASL signs.

## Features

- **ASL TO ENGLISH ALPHABET**

## Getting Started

### Prerequisites

Before you can run this application, ensure you have the following installed:

- Python (3.6+)
- Flask
- Deep learning libraries (e.g., TensorFlow, PyTorch)

### Installation

1. Clone this repository to your local machine:
2. Make sure that you have a working web cam available.
3. Open your terminal/cmd and go to the cloned repository.
4. Type "Python app.py" and run.
5. Open your browser and go to "http://127.0.0.1:5000/"
